## 🚀 Deployment Instructions for `kakutixyz-ai/chika-privacy`

1. Open your repository: [https://github.com/kakutixyz-ai/chika-privacy](https://github.com/kakutixyz-ai/chika-privacy)
2. Upload [`index.html`](./index.html) from this directory to the root of your `chika-privacy` repository;
3. Go to **Settings** -> **Pages** -> **Branch**, select `main` (root `/`), and click **Save**;
4. Your official Privacy Policy URL will be live at:
   ```
   https://kakutixyz-ai.github.io/chika-privacy/
   ```
